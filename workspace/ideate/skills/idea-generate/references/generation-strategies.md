# Generation Strategies

Use only the strategies relevant to the current topic.

## Gap-driven

Generate ideas from explicit limitations or open questions in the source material.

## Contradiction-driven

Generate ideas from conflicting conclusions across papers or experiments.

## Transfer-driven

Move a method from a nearby task, modality, or benchmark into the current task.

## Failure-driven

Generate ideas by inspecting past failed experiments and identifying what was missing.

## Ablation-driven

Generate ideas from controllable changes in the current baseline.

## Metric-driven

Generate ideas from current metric weaknesses, failure regions, or tradeoffs.

## Constraint-driven

Generate ideas that explicitly respect compute, data, or implementation limits.
